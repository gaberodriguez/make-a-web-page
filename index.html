Lesson 1: The Basics

The Web

The web consists of a bunch of computers communicating and connected together. When ever someone opens up browser like chrome, firefox, and internet explorer and go to a certain website like facebook. com they request this page through the HTTP protocol from a server so the browser can no display that page.

HTML

HTML stands for Hypertext Markup Language. There are many different content types that can be viewed through a browser but the majority is going to be HTML. HTML documents contain two types of information which is text and visual (how it looks).

Tags and Elements

HTML documents conisit of HTML elements. When we write HTML we tell the browsers the type of each element by using HTML tags.

An element is a group of tags, for example some elements are comprised of an opening tag, contents, and a closing tag. like b and /b are tags to make the contents between them bold.There are many different tags.

  • "em"- Italics
  • "a"- Anchor
  • "p"- Paragraph

Anchor's are different in that it contains an "Attributes".Tags can have multiple attributes. The Anchor tag in example can have href as the attribute name which is used to make links. Like this one. This link will take you google when you click on the word "one".

Images

Images are a tag as well they pull an image from a server where the picture is stored and display them on the browser. When you create an Image tag you need to write the address to the picture after "src" which stands for source, we also need an alternate text to display when the image can not be pulled. Which comes after "alt". Because the image has no content it does not need a closing tag.

Inline vs Block

Inline would be text in line and we can use the "br" tag is like hitting enter in Word. And Block is things like the "p" which would create a paragraph and what this does is it creates a block of space around the text within the "p" tags.

span / div

These tags are both noraml tags they both contain text. The only difference it that "span" is inline and "div" is block. These are both used to adding styles to your documents.

Computers are stupid

Computers are stupid because we have to tell them exatcly the right format. They take everything literally. Which makes things complicated when someone makes a single error in code. Because the computer will not tell you what's wrong or where the error is.

CSS

CSS stands for Cascading Style Sheets- style sheet language used for describing the look and formatting of a document written in a markup language.

CSS is used to add style to our webpage. Just like in the example of the house its like adding color to the walls. So we can add color or different font size or styles to our webpage.

Avoiding Repetition

Avoid Repitition: Avoinding repetion is very important. It will lead to less clutter it will make it easier when you want to change things or attributes. Which might causes errors when one little mistake is made when rewriting the same code many times.

Consistency is vital: We are creatures of pattern and order. Its a whole lot easier to read or look at and understand a page when it is organized and looks neat and has the same style. For example I put every paragraph in this webpage in the same class. So now if I want to change they style of paragraphs I can do so by just a couple of steps. Now if I have reapeated code and made the paragraphs all different that would take a whole lot longer to adjust all my paragraphs.
